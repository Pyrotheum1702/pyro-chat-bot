# Selected Work

## TON Poker — Real-time multiplayer poker (2024)
Multiplayer poker with AI opponents and TON blockchain integration: a complete
poker engine (full Texas Hold'em rules, hand evaluation, betting rounds),
intelligent bot behavior for solo play, and real-time multiplayer on Colyseus.
Stack: Cocos Creator, Node.js, Colyseus, TON Blockchain, MongoDB.

## DragonSlither / SnakeTon — Web3 PvP (Xtel, 2024)
Real-time PvP slither-style game with blockchain-integrated deposits and
withdrawals, serving 1M+ users. Centralized authoritative server for cheat
prevention; custom matchmaking and lobby systems sustaining 100–300 concurrent
users; a full backend financial layer (crypto transaction pipelines, asset
ledgering) and player economy. Stack: Cocos Creator, Node.js, MongoDB, Redis,
Colyseus.

## NeoHash · Pirate Kings · Crazy Wheel — Real-time titles (Arklab)
Web games and Telegram Mini Apps operated at scale (millions of users), with
leaderboards, ranking-based rewards, shops, task systems, payments, and full
in-game economies.

## EcoDolphin — Hackathon project (2024)
Won 3rd place at the MEME.ZKWasm Hackathon hosted by Delphinus Lab in Bangkok,
Thailand.

## Casual & mobile games
3 casual puzzle games on Android / iOS with Unity (Screw Puzzle, Ghost Jam,
Monster Escape), plus 10+ promotional mini web games and earlier mobile titles.
